# Enclosure

!!! note "Coming soon — Printables"
    A 3D-printed enclosure is planned. Print files will be published on **Printables** rather than stored in this software repo. This page will link to the Printables model once it is available.

---

## Design intent

- Houses the PCB, TSL2591 sensor (exposed to sky through a clear aperture), and GPS module (optional)
- Weather-resistant design for permanent outdoor mounting
- Designed around the custom PCB footprint
- Tool-free lid removal for sensor cleaning

---

## Planned print specs

These are targets — final values will be published with the Printables model.

| Parameter | Target |
|-----------|--------|
| Material | PETG or ASA (outdoor UV resistance) |
| Layer height | 0.2 mm |
| Infill | 30–40% |
| Supports | Required for lid lip |
| Heat-set inserts | M3, lid corners |
