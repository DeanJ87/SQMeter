# RG-15 Rain Sensor

The Hydreon RG-15 is an optical rain gauge. SQMeter is an open-source alternative to commercial all-sky weather stations such as the AAG CloudWatcher. Rain detection is a core part of that role — it lets you automate observatory roof or dome closures, trigger alarms, and abort imaging sequences before your gear gets wet. The RG-15 is an optional peripheral; the sky quality functions work without it.

---

## Specifications

Taken directly from the Hydreon RG-15 manual — no values inferred.

| Spec | Value |
|------|-------|
| Supply voltage | 5–35 VDC (J1 or J2 pin 2) |
| Alternative supply | 3.3 V via J2 pin 8 — **avoid unless supply is ultra-stable; overvoltage destroys the device** |
| Current draw | 110 μA nominal (dry), 2–4 mA when raining |
| Serial interface | 3.3 V TTL (directly ESP32-compatible — no level shifter needed) |
| Default baud rate | 9600 |
| Output resolution | 0.01 in / 0.2 mm (low) or 0.001 in / 0.02 mm (high) |
| Operating temp | −40 °C to +60 °C |

## Connectors

The RG-15 has two connectors:

**J1** — power and open-collector output (bucket-tip style):

| Pin | Signal |
|-----|--------|
| OUT | Open collector output, pulled to GND per tip |
| V+  | 5–35 VDC supply |
| GND | Ground |

**J2** — 8-pin 0.1″ field used for serial communication and optional power:

| Pin | Signal |
|-----|--------|
| 1 | GND |
| 2 | V+ (5–35 VDC) |
| 3 | OUT (same as J1 OUT) |
| 4 | Serial OUT — sensor transmits data here |
| 5 | Serial IN — send commands here |
| 6 | TB IN — bridge to GND to enable external tipping bucket |
| 7 | MCLR |
| 8 | V+ 3.3 V (unstable supply causes false indications — prefer J1/J2-2) |

SQMeter uses J2 pins 1, 2, 4, and 5 for serial communication.

---

## Wiring

Connect J2 to the ESP32. The sensor's Serial OUT (pin 4) goes to the ESP32 RX pin, and the sensor's Serial IN (pin 5) goes to the ESP32 TX pin.

```
RG-15 J2        ESP32
  Pin 1 (GND)  ──────  GND
  Pin 2 (V+)   ──────  5V
  Pin 4 (TX)   ──────  GPIO 18 (RX)
  Pin 5 (RX)   ──────  GPIO 19 (TX)
```

!!! note "Level shifting"
    The RG-15 operates at 3.3 V TTL. ESP32 GPIO pins are 3.3 V. No level shifter is required.

## RS232 Serial Protocol

The RG-15 uses 3.3 V TTL serial. All output lines end with `\r\n`. Commands require only `\n` to be processed.

### Commands

| Command | Description | Example response |
|---------|-------------|-----------------|
| `A` | Read accumulation since last message | `Acc 0.000 in` |
| `R` | Read all available data | See output format below |
| `K` | Restart device | outputs header |
| `B [code]` | Set baud rate (or query current) | `Baud 9600` |
| `P` | Set polling-only mode | `p` |
| `C` | Set continuous mode (output on change) | `c` |
| `H` | Force high resolution | `h` |
| `L` | Force low resolution | `l` |
| `I` | Force imperial units | `i` |
| `M` | Force metric units | `m` |
| `S` | Use DIP switch for resolution and units | `s` |
| `O` | Reset total accumulation counter | *(no response)* |
| `X` | Enable external tipping bucket input | `x` |
| `Y` | Disable external tipping bucket input | `y` |

**Baud rate codes** (for `B` command): 0=1200, 1=2400, 2=4800, 3=9600 (default), 4=19200, 5=38400, 6=57600

### `R` response format

```
Acc 0.000 in, EventAcc 0.000 in, TotalAcc 0.000 in, RInt 0.000 iph
```

| Field | Description |
|-------|-------------|
| `Acc` | Accumulation since the last `R` message |
| `EventAcc` | Total accumulation for the current rain event |
| `TotalAcc` | Total accumulation since the last `O` reset |
| `RInt` | Rain intensity (in per hour, or mm/hr in metric mode) |

**Rain event definition:** an event starts when the first drop is detected and ends 60 minutes after the last detected drop. `EventAcc` is cleared when the event ends.

### Diagnostic output keywords

| Keyword | Meaning |
|---------|---------|
| `LensBad` | Lens cannot get sufficient light — clean or inspect lens |
| `EmSat` | Emitter saturated — diagnostic indicator |
| `Reset N/M/W/O/U/B/D` | Reset reason on power-up |

## Configuration

```json
"rain": {
  "enabled": true,
  "rxPin": 18,
  "txPin": 19,
  "baudRate": 9600,
  "mode": "continuous",
  "resolution": "switch",
  "units": "metric"
}
```

| Field | Values | Description |
|-------|--------|-------------|
| `enabled` | `true` / `false` | Enable RG-15 polling |
| `rxPin` | GPIO number | ESP32 RX ← RG-15 J2 pin 4 |
| `txPin` | GPIO number | ESP32 TX → RG-15 J2 pin 5 |
| `baudRate` | 1200 – 57600 | Must match RG-15 current baud setting |
| `mode` | `"polling"` / `"continuous"` | Polling = query on interval; continuous = output on change |
| `resolution` | `"high"` / `"low"` / `"switch"` | Override resolution or defer to DIP switch |
| `units` | `"metric"` / `"imperial"` / `"switch"` | Override units or defer to DIP switch |

## Sensor Placement

- Mount outdoors with a clear, unobstructed sky view directly above the sensor aperture
- Keep away from roof overhang drip lines — water dripping from above is not rain
- Mount level so water drains correctly off the lens
- Avoid locations prone to debris, bird activity, or spray
