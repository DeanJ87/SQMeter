# SQMeter

Modern ESP32-based dark sky monitoring system for measuring sky brightness, environmental conditions, and light pollution.

SQMeter provides live SQM, NELM, Bortle, lux, temperature, humidity and pressure readings through a built-in web interface, REST API, WebSocket updates and optional MQTT publishing.

## Documentation

Full setup, wiring, API, MQTT, development and troubleshooting documentation is here:

**https://deanj87.github.io/SQMeter/**

## Highlights

- ESP32-based firmware
- TSL2591 light sensor support
- BME280 environmental sensor support
- SQM, NELM and Bortle calculations
- Built-in responsive web dashboard
- REST API and WebSocket live updates
- Optional MQTT publishing
- OTA updates
- Captive portal Wi-Fi setup
